# Security

## 지원 범위

이 저장소는 포트폴리오와 학습 목적의 공개 코드입니다. 실제 운영 데이터, 인증 정보, 모델 가중치, 사용자 업로드 문서는 저장소에 포함하지 않습니다.

## 민감 정보 관리

- `.env` 파일은 커밋하지 않습니다.
- API 키, DB 비밀번호, SMTP 계정은 환경 변수로만 주입합니다.
- 운영 데이터나 사용자 문서는 테스트 fixture로도 올리지 않습니다.
- 스크린샷을 올릴 때는 개인 정보와 내부 주소를 제거합니다.

## 취약점 제보

이슈에는 비밀번호, 토큰, 운영 URL 같은 민감 정보를 남기지 말아 주세요. 공개 이슈로 설명하기 어려운 내용은 저장소 소유자에게 별도로 연락해 주세요.

## 배포 전 확인

- 기본 비밀번호 제거
- CORS 허용 범위 제한
- DB 계정 권한 최소화
- 로그에 질문 원문과 개인정보가 남는지 확인
- 업로드 파일 크기와 확장자 제한
